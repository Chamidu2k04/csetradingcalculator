# CSE Trading Calculator

This is a **mobile-friendly share trading calculator** created to help beginners and learners quickly calculate:

- Invested amount  
- Trading fees (1.2% default)  
- Shares bought  
- Net profit or loss  

The tool is designed for **educational purposes** and demonstrates how trading fees affect overall profit or loss. It works with the **Colombo Stock Exchange (CSE)** or any other stock market where similar fees apply.

## Features

- Simple, clean HTML interface  
- Mobile-optimized layout for easy use on smartphones  
- Color-coded results: **green** for profit, **red** for loss  
- Displays invested amount, buy/sell fees, final amount, and profit/loss  
- Fully static, no backend required  

## Usage

1. Enter your **buy amount**, **buy price per share**, and **sell price per share**.  
2. Click **Calculate**.  
3. View results instantly with clear profit/loss display.
